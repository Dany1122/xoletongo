Atributos Base del Modelo "Producto" (Haz clic para expandir/contraer)

Los siguientes atributos son fijos y estarán presentes en todos los productos de todas las empresas. Los atributos personalizados que definas a continuación se añadirán a esta estructura base.

  • nombre: Texto para el nombre del producto.
  • descripcion: Texto largo para la descripción.
  • precio: Campo numérico para el costo.
  • sku: Código de inventario único.
  • stock: Cantidad de unidades disponibles.
  • imagen: Archivo de imagen principal.
  • activo: Booleano (Sí/No) para visibilidad.
  • perecedero: Booleano (Sí/No) si tiene caducidad.
  • fecha_caducidad: Campo de fecha.
  • categoria: Relación con una categoría.
