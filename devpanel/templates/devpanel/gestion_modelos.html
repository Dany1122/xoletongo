Atributos Base del Modelo "Producto" (Haz clic para expandir/contraer)

Los siguientes atributos son fijos y estarán presentes en todos los productos de todas las empresas. Los atributos personalizados que definas a continuación se añadirán a esta estructura base.

  • nombre: Texto para el nombre del producto.
  • descripcion: Texto largo para la descripción.
  • precio: Campo numérico para el costo.
  • sku: Código de inventario único.
  • stock: Cantidad de unidades disponibles.
  • imagen: Archivo de imagen principal.
  • activo: Booleano (Sí/No) para visibilidad.
  • perecedero: Booleano (Sí/No) si tiene caducidad.
  • fecha_caducidad: Campo de fecha.
  • categoria: Relación con una categoría.

Atributos Base del Modelo "Servicio" (Haz clic para expandir/contraer)

Los siguientes atributos son fijos y estarán presentes en todos los servicios de todas las empresas. Los atributos personalizados que definas a continuación se añadirán a esta estructura base.

  • titulo: Texto para el título del servicio.
  • descripcion: Texto largo para la descripción detallada.
  • servicio: Relación con el tipo de servicio.
  • costo_por_persona: Campo numérico para el precio por adulto.
  • costo_niño: Campo numérico para el precio por niño.
  • costo_con_descuento: Campo numérico para precio con descuento.
  • imagen_principal: Archivo de imagen principal.
  • duracion: Texto para especificar duración.
  • restricciones: Texto para condiciones y restricciones.
